# Chained Together

Chained Together is a Fabric mod for Minecraft 1.16.1 that allows chaining two entities together with visual and
movement-linked behavior.

## Features

- Chain any pair of entities: players, mobs, or others
- Entities are forced toward each other if they get too far apart
- Works in both directions, no "leader"
- Chain distance is configurable
- Full client-server sync

## Commands

`/chain <target>  `
Chain yourself to the specified entity

`/chain <target1> <target2>  `
Chain two entities together

`/unchain  `
Unlink from all currently chained entities

`/chaindistance <value>`
Set the maximum allowed chain length

## Requirements

- Minecraft 1.16.1
- Fabric API
- Fabric Loader
